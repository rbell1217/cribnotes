FIRESTORE DATA MODEL
================================================================================

Collections:
  families/
    └── {familyId}/
        ├── name: string
        ├── inviteCode: string (6 chars, unique)
        ├── parentIds: array
        ├── sitterIds: array
        ├── createdAt: timestamp
        ├── children/ (subcollection)
        │   └── {childId}/
        │       ├── name, age, avatar
        │       ├── createdAt: timestamp
        │       ├── careGuide/ (subcollection)
        │       │   └── sections (document)
        │       │       ├── emergencyContacts: array
        │       │       ├── dailySchedule: array
        │       │       ├── meals: array
        │       │       ├── napsBedtime: array
        │       │       ├── diapersPotty: array
        │       │       ├── safetyTips: array
        │       │       ├── locations: array
        │       │       ├── tvEntertainment: array
        │       │       ├── carTravel: array
        │       │       ├── activities: array
        │       │       └── medicalInfo: array
        │       ├── checklists/ (subcollection)
        │       │   └── {checklistId}/
        │       │       ├── title: string
        │       │       ├── items: array (with done flag)
        │       │       ├── createdAt: timestamp
        │       │       └── completedAt: timestamp
        │       └── photos/ (subcollection)
        │           └── {photoId}/
        │               ├── url: string
        │               ├── caption: string
        │               ├── createdAt: timestamp
        │               └── uploadedBy: uid
        └── messages/ (subcollection)
            └── {msgId}/
                ├── from: uid
                ├── fromName: string
                ├── text: string
                ├── timestamp: timestamp
                └── read: boolean

  users/
    └── {uid}/
        ├── email: string
        ├── name: string
        ├── role: string (parent | babysitter)
        ├── familyId: string (reference)
        ├── avatar: string (URL or null)
        └── createdAt: timestamp
